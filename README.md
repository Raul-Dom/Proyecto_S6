# Proyecto_S7
# Vehicle Sales Dashboard

This is a simple Streamlit web application created as part of the TripleTen Data Science Bootcamp (Sprint 7: Software Development Tools).

The goal of this project is to practice:

- Working with virtual environments  
- Using Git/GitHub  
- Performing basic Exploratory Data Analysis (EDA)  
- Building a small interactive dashboard with Streamlit  
- Deploying a web app to the cloud (Render)

---

## Features

The application includes:

### Header  
A main title describing the dashboard.

### Histogram  
A Plotly interactive histogram showing the distribution of vehicle odometer readings.

### Scatter Plot  
An interactive scatter plot showing the relationship between vehicle odometer and price.

### Buttons  
Two buttons that allow users to display the charts on demand.
