# Análisis del Mercado de Videojuegos – Proyecto Sprint 6

Proyecto realizado como parte del bootcamp de Ciencia de Datos de TripleTen.  
El objetivo es analizar un dataset real de ventas de videojuegos para identificar patrones comerciales y tendencias relevantes para la tienda digital **Ice**.

## Contenido del análisis
- Limpieza y preparación del dataset.
- Exploración de ventas por plataforma, género y región.
- Evaluación del ciclo de vida de consolas.
- Impacto de las reseñas de usuarios y críticos en las ventas.
- Comparación de medias mediante pruebas estadísticas (t-test).
- Conclusiones y recomendaciones basadas en los hallazgos.

## Tecnologías utilizadas
- Python  
- Pandas  
- Matplotlib / Seaborn  
- SciPy  
- Jupyter Notebook  

---

Este proyecto demuestra habilidades en análisis exploratorio de datos, estadística y visualización, integrando resultados en conclusiones de negocio.
